# 标题

**标题**用#、##、###表示一级二级三级标题
加粗用两个\*包起来，斜体用一个\*包起来，斜体加粗则用三个，删除线用两个~~
引用，用多个>
分割线：三个或者以上的-或者\*都可以
**图片**：![图片alt](图片地址 "图片title")
超链接：[超链接名](超链接地址 "超链接title")

列表：用-+\*任何一种都可以
有序列表：数字加点
列表嵌套：上一级和下一集之间敲三个空格

表格：
表头|表头|表头
---|:--:|---:
内容|内容|内容
内容|内容|内容
内容|内容|内容

代码：
当行代码用一个反引号包起来
代码块用三个反引号包起来，并且两边的反引号都是独占一行
```
   代码...
   代码...
   代码...
```
